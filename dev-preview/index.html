Character Generator WordPress Plugin

Development Preview

This is a WordPress plugin — it requires a WordPress installation to run fully. This preview is here to confirm the build environment is working.

To use the plugin:

  1. Copy the plugin folder into your WordPress wp-content/plugins/ directory.
  2. Activate it from the WordPress admin panel.
  3. Add the shortcode [character_generator] to any page.

To rebuild assets after making changes, run: npm run build
